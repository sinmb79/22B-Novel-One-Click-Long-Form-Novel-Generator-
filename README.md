핵심 기능 / Core features

- Author DNA validation and deterministic plot architecture
- Chapter generation with runtime provider routing
- SQLite memory tracking and rule-based review
- Markdown, TXT, JSON, EPUB, PDF export
- Next.js dashboard for project browsing
